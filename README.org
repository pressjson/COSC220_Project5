*** How?

With [[https://github.com/ollama/ollama-python][Ollama Python Library]]. I ask the model to do the conversion, and it gives me an image. Boom. Easy.

#+begin_src python
from ollama import chat, ChatResponse

response: ChatResponse = chat(model='gemma3', messages=[
    {
        'role': 'user',
        'content': 'Convert this image from grayscale to color.',
        'images': image_path,
    }
])

with open('output.png', 'wb') as f:
    f.write(response.image)
